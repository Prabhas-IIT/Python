# Scientific Computing Projects (Python)

This repository contains my solutions to the **Scientific Computing with Python** certification projects. These projects focus on strengthening core Python programming concepts including functions, object-oriented programming (OOP), string manipulation, simulations, and algorithmic problem solving.

## Projects

### 1. Arithmetic Formatter
Formats arithmetic expressions vertically in the same style taught in elementary mathematics.

#### Features
- Supports addition and subtraction
- Properly aligns operands and operators
- Optional display of computed answers
- Input validation and meaningful error messages

**Concepts Used**
- String formatting
- Input validation
- Lists
- Functions

---

### 2. Time Calculator
Calculates the resulting time after adding a duration to a given start time.

#### Features
- 12-hour clock support (AM/PM)
- Handles day rollover
- Calculates multiple-day differences
- Optional weekday calculation
- Correct formatting for next day and future days

**Concepts Used**
- Arithmetic
- Conditional statements
- String manipulation
- Modular arithmetic

---

### 3. Budget App
A budget management application built using Object-Oriented Programming.

#### Features
- Deposit and withdrawal tracking
- Category transfers
- Balance management
- Spending validation
- ASCII spending percentage chart

**Concepts Used**
- Classes & Objects
- OOP
- Lists & Dictionaries
- Data visualization using text

---

### 4. Polygon Area Calculator
Implements Rectangle and Square classes demonstrating inheritance and polymorphism.

#### Features
- Area, perimeter and diagonal calculations
- ASCII visualization of shapes
- Shape fitting calculations
- Inheritance between Rectangle and Square

**Concepts Used**
- Object-Oriented Programming
- Inheritance
- Method overriding
- Geometry

---

### 5. Probability Calculator
Estimates probabilities using Monte Carlo simulation.

#### Features
- Random sampling without replacement
- Configurable experiments
- Probability estimation through repeated simulations

**Concepts Used**
- Randomization
- Simulation
- Probability
- Statistical approximation
